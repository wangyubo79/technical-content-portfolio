# AI/LLM Technical Content Portfolio

A curated portfolio of AI/LLM developer education work across Mandarin and English tutorials, workshops, blogs, videos, and code samples. The content focuses on helping developers move from first AI concepts to hands-on GenAI applications, agentic developer tools, and production-ready AI systems.

## Featured Technical Content

### 1. Computer Vision with Transformers on Amazon SageMaker

- Formats: [video](https://www.youtube.com/watch?v=UpFnghPlR_Y) and [GitHub code](https://github.com/wangyubo79/vision-transformer-sagemaker)
- Explains Vision Transformer architecture and how to run it on Amazon SageMaker.
- Includes runnable code that developers can inspect, adapt, and extend.
- Audience: AI engineers, ML builders, and developers learning transformer-based computer vision.

### 2. Rapid Generative AI Application Development with Amazon Bedrock

- Format: [hands-on lab](https://events.amazoncloud.cn/labs/cloudlab-cloud9-call-bedrock-api-and-model-fine-tune)
- Guides developers from foundation model access to a working GenAI application pattern.
- Highlights how builders can use Amazon Bedrock with foundation models such as Anthropic Claude and Meta Llama.
- Focuses on practical onboarding, API usage, and application development flow.
- Audience: cloud developers and AI builders starting with GenAI applications.

### 3. Prompt Chaining Workflows with Amazon Bedrock and Step Functions

- Format: [hands-on lab](https://events.amazoncloud.cn/labs/cloudlab-prompt-chaining-based-on-bedrock-and-stepfunction)
- Demonstrates orchestration patterns for multi-step LLM workflows.
- Helps developers reason about agent-like systems, workflow design, and production control flow.
- Audience: developers building LLM applications, workflow automation, and AI agents.

### 4. LLMOps for Production Generative AI Applications

- Format: [technical blog](https://mp.weixin.qq.com/s?__biz=MzkxMjM2MDIyNQ==&mid=2247641536&idx=1&sn=30cd5dac7b4184acd2d66ab537379d1b&chksm=c018925192af57c5c4da6bf18de46554fac7f7fe720af65f77ff17e8f2d9dba535d3358fa237)
- Explains how GenAI applications move from prototype to production operation.
- Covers the operating model developers need for iteration, deployment, and reliability.
- Audience: AI application engineers and teams operationalizing LLM-powered products.

### 5. Training and Serving Large Language Models on Kubernetes

- Format: [technical blog](https://mp.weixin.qq.com/s?__biz=Mzg5Mzg1NDc2NQ==&mid=2247493909&idx=1&sn=4b7c5cf57e8bf3610c5fcdee0a430a52&chksm=c13fb4cd0a27f365ce3ef6a998bf6e74457a214de688a13f8f195ca120528ccb3ead32cf1a24)
- Bridges LLM training and serving with cloud-native deployment practices.
- Connects AI infrastructure concepts to Kubernetes-based production systems.
- Audience: platform engineers, ML infrastructure teams, and developers deploying AI workloads.
